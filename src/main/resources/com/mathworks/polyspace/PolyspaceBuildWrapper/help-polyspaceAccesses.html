Polyspace Access

In this configuration panel, Polyspace Access configurations can be provided. For each configuration, the following information must be provided:

  • The name of the configuration. It will be referenced in the Build Environment section of the Freestyle Project
  • The protocol of the server, i.e. http or https
  • The hostname of the server
  • The port of the server, typically 9443.

Note that:

  • This is not mandatory to have at a Polyspace Access configuration to be able to use the Polyspace Jenkins Plugin.
  • Polyspace Access has been introduced in Polyspace R2019a.
